What are the questions that participants should be ranked on? You can create a game with only custom questions. Or you can create a game with only predefined questions. Or you can make a combination of predefined and custom questions. When going for predefined questions, it helps to pick a setting. You don't want to play a family game and ask "who is the biggest slut". You don't want to play a game with colleagues and ask "who's farts stink the worst". The setting determines the questions. Pick a setting, and the predefined questions will be suggested accordingly.
